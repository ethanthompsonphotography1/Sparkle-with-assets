# Asset Library Summary

This directory contains the processed game assets for the Sparkle project, organized by category. All sprites have had their backgrounds removed while preserving the signature thick white sticker-style borders.

## Asset Counts by Category

| Category | Count | Description |
| :--- | :--- | :--- |
| Characters | 14 | Mascots, Beat-Bugs, Unicorns, and other character sprites. |
| Environments | 7 | Full-frame 16:9 background environments. |
| UI & Care Kit | 4 | Navigation buttons and interactive Care Kit items. |
| Closet | 1 | Modular clothing overlays and closet assets. |
| Rhythm | 2 | Rhythm game elements like stars and music notes. |

## Directory Structure

- `/assets/images/characters/`: Character and companion sprites with transparent backgrounds.
- `/assets/images/environments/`: 16:9 environment backgrounds (full-frame).
- `/assets/images/ui/`: UI navigation icons and interactive Care Kit items.
- `/assets/images/closet/`: Sparkle Closet modular overlays.
- `/assets/images/rhythm/`: Rhythm game elements and effects.

## Processing Details
- **Background Removal:** Performed on all sprites and UI elements.
- **Border Preservation:** Thick white sticker-style borders have been kept intact as per the visual design system.
- **Format:** All assets are saved as transparent PNGs (where applicable).
